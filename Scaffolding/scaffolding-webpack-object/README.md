## 脚手架

使用
// 拷贝项目到本地
npm install -g struct-cli

Usage: struct [options]

Options:

-V, --version  output the version number
-h, --help     output usage information
Commands:

list        显示所有模板
add         添加模板
delete      删除模板
init        用一个模板生成项目
help [cmd]  display help for [cmd]
